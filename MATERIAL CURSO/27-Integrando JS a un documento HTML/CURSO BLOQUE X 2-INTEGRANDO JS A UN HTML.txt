Al igual que con CSS y su link rel a una hoja externa, lo mejor sera crear una carpeta de JS aparte y crear un archivo externo con el cual relacionarnos. Para ello, entonces:

a) Creamos una nueva carpeta de JS
b) Creamos una hoja function.js
c) Mediante un elemento <script> en el body, usamos src para llamar a la hoja externa

<script src="js/function.js"></script>

La papa es que si yo armo un documento HTML utilizando 2 hojas externas (de CSS y JS), es un gran practica de programacion, ya que tendre una hoja limpia y ordenada, ademas de que podre reutilizar codigo.

COMO LO INTEGRAMOS A HTML Y COMO FUNCIONA UN BOTON
1) Tomemos el ejemplo del funcionamiento de un boton de w3schools:
https://www.w3schools.com/js/tryit.asp?filename=tryjs_intro_inner_html

<!DOCTYPE html>
<html>
<body>

<h2>What Can JavaScript Do?</h2>

<p id="demo">JavaScript can change HTML content.</p>

<button type="button" onclick='document.getElementById("demo").innerHTML = "Hello JavaScript!"'>Click Me!</button>

</body>
</html>

Que es lo que hace este evento? Cambia el texto que se muestra cuando haces clic en el boton Click me!

2) Ahora, vamos a profesionalizarlo un poco y vamos a trabajar con la funcion en una hoja externa:

<body>

    <p id="demo">Estamos aprendiendo JavaScript</p>

    <button type="button" id="boton">Hacer clic aquí</button>

    <script src="js/boton.js"></script>
    
</body>
</html>

Lo que hicimos fue crear una funcion y eventos en la hoja de js usando las id"boton" y "demo" como referencias:

document.getElementById("boton").onclick = function () {
    console.log("Capturamos el evento click");
    document.getElementById("demo").innerHTML = "Estamos probando nuestro primer evento"
}

-document.getElementById te busca a la ID del boton y
-.onclick es un EVENTO (los eventos arrancan con puntos) que generara una funcion
de console.log que de ese mensaje en la consola
-document.getElementById("demo").innerHTML = hara que se cambie el texto anterior por el de este otro al hacer clic

Inclusive puedo customizar como se ven los parrafos "demo" si meto dentro de css el cambio, o sea, puedo laburar en 3 partes al mismo tiempo.
