BLOQUE III - SELECTORES, TIPOGRAFIAS, ID Y CLASS
1) SELECTORES, ID Y CLASS
Recordemos que, en CSS, los SELECTORES son los elementos cuyas propiedades vamos a modificar mediante STATEMENTS que se compondrán de diferentes VALUES. Pero, sin embargo, pueden tomar varios tipos según qué o a quiénes modifiquen.

a. Selector UNIVERSAL (*)
* {
  text-align: center;
  color: blue;
}
Con el uso del asterisco, vos marcas que TODOS los elementos HTML de tu pagina van a estar modificados con esos valores. Si tengo un h1 y un p, todos quedaran bajo esto. PERO aca entra la very definition de CSS, "hoja de estilos en CASCADA". Si defino un valor universal, todos los elementos tendran esos valores, pero VALEN MAS los que estan definidos por dentro.

Si tengo un <p> definido, valdrán en todos los <p> estos valores y no los del *

p {
    color: red;
    border: 1px dotted green;
    padding: 30px 15px 25px 20px
}


b. AGRUPAR selectores
Si hay elementos (selectores) los cuales quiero que compartan propiedades, los separo, como habiamos visto anteriormente, con comas. Entonces:
h1, h2, p {
  text-align: center;
  color: red;
}
Los títulos h1 y h2 y los p estarán alineados y seran todos rojos.


c. Atributo ID
Si profundizamos un poco y queremos que un elemento solo de un determinado tipo tenga propiedades diferentes de los demas, ahi debemos recurrir al atributo ID. El selector ID usa el atributo de un elemento HTML especifico para seleccionar un unico elemento. El ID es unico para cada pagina y para cada elemento en particular y se introduce mediante un #nombre del ID. NO PUEDE EMPEZAR CON UN NUMERO, sí tenerlo dentro (#para1).

Entonces, para definirlo, se trabaja en dos frentes: HTML y CSS.
-Primero, se define el ID en el código:
<p id="mi_parrafo">Texto</p>
-Y luego, en CSS, marcamos sus propiedades, abriendo con #
#mi_parrafo {
    color: brown
    font-size: 23px
}


d. Atributo CLASS
Dado que es una realidad que en un proyecto con varias paginas, no se si vamos a querer que todos los <> se vean identicos; podemos, por ejemplo, querer que cada <h1> de cada pagina que integre el proyecto tenga su estilo, tenemos a mano para ello el atributo CLASS.

El atributo CLASS, al igual que el ID, se define en nuestros dos frentes, pero a la inversa:
-Primero, se define la clase dentro de CSS, abriendo con un (.)
.estaesmiclase {
  font-weight: bold;
  color: lightblue;
}
-Y luego vamos a HTML e identificamos con dicha clase a todos los elementos que queremos que estén marcados con ella:
<p class="estaesmiclase">Párrafo con estilo por CLASS</p>

Existen un montooon de reglas de cómo los selectores pueden comportarse segun clases, todas acá: https://htmlcheatsheet.com/css/. Por ejemplo, podemos querer que todos los h1 que esten dentro de un section tengan determinado estilo, debere marcar:
section h1 {
    font-weight: 100 px
}


NOTA: en el caso de un documento de UNA pagina sola (como nos va a tocar a nosotros), podemos trabajar con selectores de a grupos porque la hoja CSS no va a ser una locura pero, para el caso de proyectos mas grandes, de varios HTML, es mas recomendable echarle mano a los atributos ID y CLASS, ya que va a ser mas prolijo y definido que pelear contra propiedades universales de todos los elementos de un grupo.


2) TIPOGRAFIAS - FONTS
https://www.w3schools.com/css/css_font.asp
De mas esta decir que la eleccion de las tipografias es super importante, por los estilos y mensajes que vos queres dar en general y con cada seccion. Pero hay que entender algo. Si bien Chrome viene con 5 familias genericas de fuentes, las cuales son las Serif, Sans Serif, Monospace, Cursive y Fantasy (que contienen diversas y muy conocidas letras dentro), existe la posibilidad de importar tipografias externas desde Google Fonts (ya sea te las pida el cliente o te las sugiera el diseñador web).

*Google Fonts*
Con esta herramienta, lo que tenemos es un banco de fuentes de codigo abierto las cuales podemos probar en su totalidad con el texto que necesitamos y ver como quedan, ademas de ver sus diferentes variantes en familia e importar todo.
-Entramos a la pagina, buscamos la fuente que queremos
-Ingresamos texto y elegimos al costado derecho la variante especifica
Ejemplo: Roboto en Thin 100 25 px
Y arriba a la derecha, en Selected Family, tenemos el codigo para introducirlo en el <head>, pues el procedimiento tiene 2 partes:

a. Copiar el link rel de la tipografia e insertarlo en el <head> (HTML)
<link rel="preconnect" href="https://fonts.googleapis.com">
<link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
<link href="https://fonts.googleapis.com/css2?family=Roboto:wght@100&display=swap" rel="stylesheet">

b. Y, para CSS, especificamos el atributo "font-family", que será el encargado de marcar la familia de tipografias que necesitamos para nuestro proyecto, y copiamos de mas abajo de donde copiamos lo anterior la familia, en este caso:
font-family: 'Roboto', sans-serif
